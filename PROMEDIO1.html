SESIÓN 7

Hosting y Dominio

Abril 9, 2024 LABORATORIO 2

En esta sesión pudimos ver sobre el hosting es como el alquiler de un espacio en Internet para guardar todos los archivos y datos de un sitio web. Es donde se guardan las páginas web, las imágenes, los videos y todo lo necesario para que un sitio esté disponible en línea. El dominio es como la dirección única de una casa en Internet. Es lo que escribes en la barra de dirección de tu navegador para visitar un sitio web específico, como "www.ejemplo.com". Cada sitio web tiene su propio dominio, que es como su identificación exclusiva en la web. Además, pudimos realizar las correciones del trabajo de cada estudiante en clase, agregando y quitando elementos para que nos ayuden a mejorar la página.

SESIÓN 9

Electrónica

Abril 16, 2024 LABORATORIO 2

En esta sesión, reforzamos diferentes aspectos clave. Primero, aprendimos a entregar trabajos utilizando enlaces en GitHub o Netlify. Luego, nos familiarizamos con el programa Tinkercad para llevar a cabo ejercicios con Arduino. Durante la sesión, se nos introdujo a los conceptos de electrónica básica y se explicó detalladamente qué es Arduino: una plataforma de prototipado electrónico de código abierto que combina hardware y software fácil de usar. Se destacó que Arduino consiste en una placa con un microcontrolador y un entorno de desarrollo integrado (IDE) que simplifica la escritura y carga de código en la placa. Además, se nos guió a través de un ejercicio práctico utilizando código. Para concluir, el docente ayudó a resolver nuestras dudas sobre la creación de páginas web y adelantó que en la próxima clase nos adentraríamos en la construcción de proyectos con Arduino, explorando nuevas técnicas.

SESIÓN 10

Protoboard

Abril 17, 2024 LABORATORIO 2

En esta sesión, nos adentramos en el mundo de Arduino. Nos proporcionaron los materiales necesarios, como LEDs y protoboard, para llevar a cabo diversos ejercicios. También nos instruyeron sobre la instalación del programa Arduino 1.8 para escribir y cargar códigos. Se nos explicó la importancia de entender los conceptos de ánodo y cátodo, especialmente en relación con componentes electrónicos como los LEDs. El ánodo, la patita larga, es el terminal positivo, mientras que el cátodo, la patita corta, es el terminal negativo. Además, se nos orientó sobre el uso de resistencias y la diferencia entre las fuentes de voltaje de 3.3V y 5V, así como la conexión a GND. Este es solo el comienzo de nuestro viaje en el mundo de Arduino, y esperamos seguir aprendiendo juntos en las próximas sesiones. ¡Los espero!
