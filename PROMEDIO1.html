SESIÓN 7

Hosting y Dominio

Abril 9, 2024 LABORATORIO 2

En esta sesión pudimos ver sobre el hosting es como el alquiler de un espacio en Internet para guardar todos los archivos y datos de un sitio web. Es donde se guardan las páginas web, las imágenes, los videos y todo lo necesario para que un sitio esté disponible en línea. El dominio es como la dirección única de una casa en Internet. Es lo que escribes en la barra de dirección de tu navegador para visitar un sitio web específico, como "www.ejemplo.com". Cada sitio web tiene su propio dominio, que es como su identificación exclusiva en la web. Además, pudimos realizar las correciones del trabajo de cada estudiante en clase, agregando y quitando elementos para que nos ayuden a mejorar la página.
